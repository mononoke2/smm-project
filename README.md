# Social Media Management A.A 21/22

Implementazione di un tutorial volto all'uso delle API (Steamworks) di Steam, piattaforma digitale sviluppata da Valve Corporation che si occupa di distribuzione
digitale. Segue l'implementazione di un progetto di profilazione condotto sulla medesima piattaforma al fine di estrarre tramite Bag of Words i possibili giochi da suggerire per l'acquisto.
